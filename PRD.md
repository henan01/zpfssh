# ZenSSH — 产品需求文档 (PRD)

**版本：** v0.2
**日期：** 2026-03-25
**定位：** 原生 macOS SSH 终端管理工具，轻量、快速、专业

---

## 一、产品概述

### 1.1 产品目标

打造一款原生 macOS SSH 终端客户端，对标并超越 ZenTermLite，满足开发者/运维人员的日常需求。核心原则：

- **原生优先**：纯 Swift + AppKit/SwiftUI，零 WebView，零 Electron
- **极致性能**：内存占用 < 80MB（空载），响应延迟 < 16ms
- **功能完备**：覆盖日常 SSH 操作全场景

### 1.2 目标用户

- macOS 开发者、运维工程师、DevOps 工程师
- 需要同时管理多台服务器的用户
- 对工具性能和颜值有双重要求的用户

---

## 二、技术架构

```
┌─────────────────────────────────────────────────┐
│                    UI 层                         │
│         SwiftUI + AppKit (原生渲染)               │
├─────────────────────────────────────────────────┤
│                  终端模拟层                        │
│            SwiftTerm (VT100/xterm-256color)      │
├──────────────────┬──────────────────────────────┤
│    SSH 协议层     │         文件传输层              │
│  SwiftNIO SSH    │      SFTP (libssh2)           │
├──────────────────┴──────────────────────────────┤
│                  数据持久层                        │
│    CoreData (配置) + Keychain (凭据加密存储)       │
└─────────────────────────────────────────────────┘
```

**关键依赖：**

| 组件 | 库 | 说明 |
|---|---|---|
| 终端模拟 | SwiftTerm | 纯 Swift，Metal 加速渲染 |
| SSH 协议 | SwiftNIO SSH | Apple 官方维护 |
| 文件传输 | libssh2 | SFTP 上传/下载 |
| 数据存储 | CoreData + Keychain | 配置本地化，密码加密 |

## 三、功能模块详细设计

### 3.1 服务器管理

#### 功能列表

- **服务器列表**：左侧边栏展示，支持分组/文件夹管理
- **连接配置**：
  - 主机 / 端口 / 用户名
  - 认证方式：密码 / SSH 私钥 / SSH Agent
  - 跳板机 (Jump Host) 配置
  - 自定义连接超时、Keep-Alive 间隔
- **快速连接**：⌘K 唤起命令面板，搜索服务器名快速连接
- **连接状态**：实时显示连接状态（连接中 / 已连接 / 断线重连）
- **自动重连**：网络波动后自动尝试重连，可配置重试次数

#### 服务器信息展示（连接后）
- 主机名、IP、用户名、连接时长
- 实时带宽（上行/下行）
- CPU / 内存使用率（通过后台轮询命令获取，可关闭）

---

### 3.2 终端模拟器

#### 核心能力

- **完整 VT100/VT220/xterm-256color 支持**
- **真彩色 (TrueColor / 24-bit)** 支持
- Metal GPU 加速渲染，滚动流畅无撕裂
- 字体：支持任意等宽字体，推荐 JetBrains Mono / SF Mono / Hack
- 字号：6pt ~ 72pt 自由调节，⌘+ / ⌘- 快速缩放
- 光标样式：块状 / 竖线 / 下划线，支持闪烁控制

#### 交互功能

- 鼠标支持：滚轮翻页、选中复制、右键菜单
- 智能选中：双击选词，三击选行，⌘A 全选
- 链接识别：自动识别 URL，⌘+点击 在浏览器打开
- 回滚缓冲：可配置缓冲行数（默认 10000 行）

#### 终端输出搜索

- **⌘F** 唤起搜索栏，在当前终端缓冲区全文搜索
- 实时高亮所有匹配项，当前匹配项突出显示（不同颜色区分）
- **⌘G / ⌘⇧G** 跳转到下一个 / 上一个匹配项
- 支持**正则表达式**搜索（搜索栏切换开关）
- 支持**大小写敏感**开关
- 匹配计数显示：`第 3 / 17 项`
- 搜索时终端自动滚动到匹配位置，退出搜索后恢复滚动位置
- 搜索范围：当前可见区 / 全部缓冲区（可切换）

### 3.3 分屏管理

#### 分屏操作

| 操作 | 快捷键 |
|---|---|
| 垂直分割（左右） | ⌘D |
| 水平分割（上下） | ⌘⇧D |
| 关闭当前面板 | ⌘W |
| 切换到左侧面板 | ⌘⌥← |
| 切换到右侧面板 | ⌘⌥→ |
| 切换到上方面板 | ⌘⌥↑ |
| 切换到下方面板 | ⌘⌥↓ |
| 最大化当前面板 | ⌘⇧Enter |
| 调整面板大小 | 拖拽分割线 |

#### 分屏布局

- 支持最多 **6 个面板**同时显示
- 布局预设：1/2/3/4 等分、主从布局
- 布局可保存为模板，下次一键恢复
- **同步输入模式**：开启后，输入同步广播到所有选中面板（批量操作多台服务器）

#### 标签页

- 多标签页管理，每个标签页可包含独立分屏布局
- 标签页拖拽排序
- 标签页颜色标记（用于区分生产/测试环境）

#### 标签页标题自动命名

- 连接建立后，自动通过 `hostname` 命令获取服务器主机名，设为标签页标题
- 标题格式可配置，支持以下变量：

  | 变量 | 说明 | 示例 |
  |---|---|---|
  | `{hostname}` | 服务器返回的主机名 | `prod-web-01` |
  | `{user}` | 登录用户名 | `root` |
  | `{ip}` | 连接 IP 或域名 | `192.168.1.10` |
  | `{alias}` | 服务器列表中的别名 | `生产服务器` |
  | `{index}` | 标签页序号 | `3` |

- **默认格式：** `{alias}` → 若别名为空则 fallback 到 `{user}@{hostname}`
- 主机名获取失败时（网络延迟等）显示连接别名或 IP，获取成功后自动更新
- 支持**手动重命名**（双击标签页标题），手动命名后不再自动覆盖，右键可恢复自动命名

#### 广播命令（发送到所有标签页）

- **⌘⌥Enter** 或底部工具栏"广播"按钮：将当前输入框内容发送到**所有已连接标签页**
- 广播模式分级：
  - **全部标签页**：发送到所有 SSH 会话
  - **当前窗口标签页**：只发送到当前窗口的标签页
  - **选中标签页**：⌘+点击标签页选中多个，只广播到选中项
- 广播前弹出确认提示（可在设置中关闭确认）
- 广播状态指示：标签栏显示广播图标 📡，激活的广播目标标签页高亮边框
- **广播输入条**：⌘⌥B 唤起独立输入框，专用于广播输入，与当前终端输入分离，避免误操作
- 广播记录：广播的命令会被单独记录，与普通命令历史区分（标记 `[broadcast]`）

### 3.4 文件传输 (SFTP)

#### 文件管理器

- 独立 SFTP 面板（可拆分为侧边栏或浮动窗口）
- 双栏视图：本地文件 ←→ 远程文件
- 支持目录浏览、文件预览（文本/图片）
- 显示文件权限、大小、修改时间

#### 传输操作

- **上传**：
  - 拖拽文件/文件夹到终端或 SFTP 面板
  - 右键菜单 → 上传到当前目录
  - ⌘U 快捷键打开本地文件选择器上传
- **下载**：
  - SFTP 面板右键 → 下载到本地
  - `sz` / `rz` 协议支持（ZModem）
- **传输队列**：底部状态栏显示传输进度，支持暂停/取消
- **断点续传**：大文件传输中断后可恢复
- **批量传输**：多选文件批量上传/下载

### 3.5 快捷命令 & 命令片段

#### 命令片段库 (Snippets)

- 全局命令库，跨服务器共享
- 支持分组/标签分类（如：系统监控、Docker、Git、数据库）
- 每条命令包含：
  - 名称（用于搜索）
  - 命令内容（支持多行）
  - 描述/备注
  - 快捷键绑定（可选）
  - 参数占位符：`{{param_name}}` 执行时弹窗填入

**内置常用命令模板（可编辑）：**

```
系统监控类：
- top -c                          # 查看进程
- df -h                           # 磁盘使用
- free -h                         # 内存使用
- netstat -tlnp                   # 端口监听
- tail -f /var/log/syslog         # 实时日志

Docker 类：
- docker ps -a                    # 所有容器
- docker logs -f {{container}}    # 实时日志
- docker stats                    # 容器资源
- docker-compose up -d            # 启动服务

Git 类：
- git log --oneline -20           # 提交历史
- git status                      # 工作区状态

Nginx 类：
- nginx -t                        # 配置检查
- systemctl reload nginx          # 热重载
```

#### 快捷命令面板

- **⌘P** 唤起：模糊搜索命令名，回车执行
- 支持参数模板实时填写
- 历史命令快速访问（最近 100 条）
- 命令收藏（⭐）快速置顶

#### 命令历史

- 每个会话独立历史记录，持久化保存
- 跨会话全局历史搜索（⌘⇧H）
- 支持导出历史记录（txt / json）

### 3.8 会话与工作区

- **工作区保存**：保存当前所有标签页、分屏布局、连接状态
- **启动恢复**：启动时自动恢复上次工作区（可配置）
- **会话锁定**：长时间不操作自动锁屏（需密码解锁）
- **会话日志**：自动保存会话输出到本地文件，路径可配置

---

### 3.9 安全特性

- 所有密码/私钥存储在 macOS **Keychain**，不以明文保存
- 支持 Touch ID / Apple Watch 解锁应用
- SSH 指纹验证，首次连接提示确认（TOFU 策略）
- 会话空闲自动锁定（可配置时间）
- 敏感命令二次确认（可配置关键词，如 `rm -rf`、`drop table`）

---

### 3.10 其他实用功能

- **端口转发**：本地/远程/动态端口转发，GUI 配置
- **多路复用**：同一服务器多个会话共享一条 TCP 连接（SSH Multiplexing）
- **代理支持**：HTTP / SOCKS5 代理设置
- **时区显示**：状态栏显示服务器本地时间
- **系统通知**：长时间命令完成后推送 macOS 通知
- **拖拽上传**：文件拖到任意终端窗口触发 SFTP 上传
- **右键菜单**：选中文本 → Google搜索 / 复制 / 执行命令
- **导入/导出**：服务器配置导入导出（JSON 格式，密码加密）
